Magic Shop Generator

Price Modifiers

The below formula is used to compute the price modifier of an item based on said item's tier (none, minor, major, wondrous).

y = (B * Bm) + A*ln(x)

y = price modifier

A = constant pulled from the above table

B = constant pulled from the above table

Bm = B modifier pulled from the City Magicness table.

x = population of the city

ln() = natural log

Number of Dice

The below formula is used to compute the number of dice given an item's rarity and item's type (column headers) which is used to look up the dice variable.

Nd = log(p) * Vd + log(p) + 1

Nd = number of dice

p = population of the city

Vd = dice variable pulled from the above table

log() = base 10 log

City Wealth

The below formula is used to compute wealth modifier that is applied to the total number of any given item type that a shop will have in stock.

Sf = Sb +/- Wm

Sf = final stock of any given item

Sb = base stock of any given item

+/- = operator determined by the OPERATOR column

Wm = wealth modifier determined by rolling a single die with DIE_SIZE sides.
